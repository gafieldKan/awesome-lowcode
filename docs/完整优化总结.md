## 使用建议

### 企业应用开发

1. 使用多租户架构支持 SaaS 部署
2. 利用 RBAC 实现细粒度权限控制
3. 通过插件市场扩展功能
4. 使用 CLI 快速生成代码

### 二次开发

1. 参考 `docs/二次开发指南.md`
2. 使用 CLI 创建插件和组件
3. 利用事件钩子扩展核心功能
4. 使用调试工具进行问题排查

### 性能优化

1. 启用性能监控
2. 使用性能分析工具定位瓶颈
3. 优化慢查询和慢渲染
4. 合理设置配额限制
