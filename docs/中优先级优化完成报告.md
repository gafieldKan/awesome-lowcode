## 1. 工作流可视化编排

### 新增组件

| 组件 | 说明 | 路径 |
|------|------|------|
| FlowCanvas | 可视化流程画布组件 | `lowcode/components/WorkflowDesigner/FlowCanvas.jsx` |
| FlowCanvas.css | 流程可视化样式 | `lowcode/components/WorkflowDesigner/FlowCanvas.css` |

### 核心功能

- **节点类型**: 触发器、操作、条件、结束
- **节点样式**: 不同类型不同颜色标识
- **工具栏**: 添加触发器、添加操作、运行按钮
- **空状态**: 无节点时显示引导信息

### 触发器类型 (6 种)

```js
- manual: 手动触发
- schedule: 定时触发
- webhook: Webhook
- data_change: 数据变更
- data_create: 创建数据
- data_delete: 删除数据
```

### 操作类型 (12 种)

```js
- create_data: 创建数据
- update_data: 更新数据
- delete_data: 删除数据
- send_email: 发送邮件
- send_sms: 发送短信
- http_request: HTTP 请求
- run_script: 执行脚本
- notify: 通知用户
- approve: 审批流程
- condition: 条件判断
- delay: 延时等待
- loop: 循环操作
```

### 使用示例

```jsx
import FlowCanvas from 'lowcode/components/WorkflowDesigner/FlowCanvas'

function MyWorkflow() {
  const [nodes, setNodes] = useState([])

  return (
    <FlowCanvas
      workflow={workflow}
      nodes={nodes}
      onNodesChange={setNodes}
      readOnly={false}
    />
  )
}
```

## 2. 插件市场机制

### 新增模块

| 模块 | 说明 | 路径 |
|------|------|------|
| PluginMarket.js | 插件市场管理器 | `lowcode/plugins/PluginMarket.js` |

### 核心功能

| 功能 | 说明 |
|------|------|
| 插件注册 | 内置插件自动注册 |
| 插件发现 | 从注册中心获取插件列表 |
| 版本管理 | 检查更新、版本比较 |
| 依赖检查 | 安装前检查依赖 |
| 安装/卸载 | 完整的生命周期管理 |
| 启用/禁用 | 控制插件状态 |
| 配置管理 | 插件配置持久化 |

### 插件状态

```js
PluginState = {
  INSTALLED: 'installed',    // 已安装
  AVAILABLE: 'available',    // 可用
  UPDATABLE: 'updatable',    // 可更新
  DISABLED: 'disabled',      // 已禁用
  ERROR: 'error',            // 错误
}
```

### 市场状态

```js
MarketState = {
  IDLE: 'idle',
  LOADING: 'loading',
  READY: 'ready',
  ERROR: 'error',
}
```

### 使用示例

```js
import { pluginMarket } from 'lowcode/plugins/PluginMarket'

// 设置注册中心 URL
pluginMarket.setRegistryUrl('https://plugins.awesome-lowcode.com')

// 获取所有插件
const allPlugins = pluginMarket.getAllPlugins()

// 获取已安装插件
const installed = pluginMarket.getInstalledPlugins()

// 获取可更新插件
const updatable = pluginMarket.getUpdatablePlugins()

// 安装插件
await pluginMarket.install('custom-input')

// 卸载插件
await pluginMarket.uninstall('old-plugin')

// 启用插件
await pluginMarket.enable('custom-input')

// 禁用插件
await pluginMarket.disable('custom-input')

// 更新插件
await pluginMarket.update('custom-input')

// 检查所有更新
const updates = await pluginMarket.checkUpdates()
```

### 内置插件

| 插件名称 | 类型 | 说明 |
|----------|------|------|
| custom-input | formComponent | 自定义输入组件 |
| data-source-api | dataSource | API 数据源连接器 |
| workflow-approval | workflowNode | 审批工作流节点 |

## 4. 调试工具面板

### 新增组件

| 组件 | 说明 | 路径 |
|------|------|------|
| DebugPanel.jsx | 调试面板主组件 | `lowcode/components/DevTools/DebugPanelPanel.jsx` |
| DebugPanel.css | 调试面板样式 | `lowcode/components/DevTools/DebugPanel.css` |
| useDebugTools.js | 调试工具 Hooks | `lowcode/components/DevTools/useDebugTools.js` |
| index.js | 导出文件 | `lowcode/components/DevTools/index.js` |

### 调试面板功能

| 标签页 | 功能 |
|--------|------|
| 状态 | 显示插件数、事件数、模型数等统计 |
| 插件 | 已注册的插件列表 |
| 事件 | 事件历史记录 |
| 权限 | 角色列表 |
| 模型 | 数据模型列表 |

### 调试 Hooks

```js
// 调试模式 Hook
const { isEnabled, setDebugMode } = useDebugMode({ enabled: true })

// 事件追踪 Hook
useEventTracker(['data:create', 'data:update'], (event, ...args) => {
  console.log('事件触发:', event, args)
})

// 性能监控 Hook
const { trackRender, getRenderStats } = usePerformanceMonitor({
  trackRerenders: true,
  trackLongTasks: true,
  threshold: 50,
})

// 调试日志 Hook
const { log, warn, error } = useDebugLog('MyComponent', true)
```

### 使用示例

```jsx
import { DebugPanel } from 'lowcode/components/DevTools'
import { useDebugMode } from 'lowcode/components/DevTools/useDebugTools'

function App() {
  const [visible, setVisible] = useState(false)

  // 启用调试模式
  const { setDebugMode } = useDebugMode({
    enabled: true,
    trackEvents: true,
    trackPerformance: true,
  })

  return (
    <>
      <button onClick={() => setVisible(true)}>打开调试面板</button>
      <DebugPanel visible={visible} onClose={() => setVisible(false)} />
    </>
  )
}
```
