## 1. 多租户基础架构

### 新增模块

| 模块 | 说明 | 路径 |
|------|------|------|
| TenantManager.js | 租户管理器 | `lowcode/core/TenantManager.js` |

### 核心功能

| 功能 | 说明 |
|------|------|
| 租户创建 | 支持自定义 ID、名称、slug |
| 租户状态 | active, suspended, archived, pending |
| 租户隔离 | 独立的租户上下文 |
| 配额管理 | 用户数、模型数、存储、API 调用限制 |
| 使用量追踪 | 自动记录和统计 |
| 设置管理 | 租户级自定义配置 |

### 默认配额

```js
{
  maxUsers: 10,           // 最多用户数
  maxModels: 50,          // 最多模型数
  maxStorage: 100MB,      // 存储空间
  maxWorkflows: 20,       // 最多工作流
  maxPlugins: 10,         // 最多插件
  maxApiCalls: 10000,     // 每月 API 调用
}
```

### 使用示例

```js
import { tenantManager, createTenant, setCurrentTenant } from 'lowcode/core/TenantManager'

// 创建租户
const tenant = createTenant({
  id: 'tenant-001',
  name: '示例公司',
  slug: 'example-corp',
  quotas: {
    maxUsers: 100,
    maxModels: 200,
  }
})

// 激活租户
tenantManager.activateTenant('tenant-001')

// 设置当前租户
setCurrentTenant('tenant-001')

// 检查配额
const canCreate = tenantManager.checkQuota('maxModels', 1)
if (canCreate) {
  // 创建模型
  tenantManager.incrementUsage('maxModels')
}

// 获取配额使用率
const usage = tenantManager.getQuotaUsage()
console.log(usage)
// { maxUsers: { used: 5, limit: 10, percent: 50 }, ... }
```

### 租户状态流转

```
pending --> active --> suspended --> active
                    |
                    v
                archived
```

## 3. 性能分析工具

### 新增模块

| 模块 | 说明 | 路径 |
|------|------|------|
| Performance.js | 性能监控工具 | `lowcode/utils/Performance.js` |

### 核心功能

| 功能 | 说明 |
|------|------|
| 渲染时间追踪 | 监控组件渲染耗时 |
| 长任务检测 | 检测超过 50ms 的任务 |
| 内存监控 | Chrome 内存使用监控 |
| 自定义计时器 | startTimer/endTimer |
| 函数包装 | profile/profileAsync |
| 性能报告 | 导出性能分析报告 |

### 使用示例

```js
import {
  performanceMonitor,
  startTimer,
  endTimer,
  profile,
  profileAsync,
  useRenderMonitor,
} from 'lowcode/utils/Performance'

// 启用性能监控
performanceMonitor.enable()

// 设置阈值
performanceMonitor.setThresholds({
  render: 16,  // 16ms = 60fps
  task: 50,
  memory: 50 * 1024 * 1024,
})

// 手动计时
startTimer('loadData')
const data = await loadData()
endTimer('loadData')

// 包装函数
const optimizedFn = profile(() => {
  // 耗时操作
}, 'myFunction')

// 包装异步函数
const optimizedAsync = profileAsync(async () => {
  const data = await fetchData()
  return data
}, 'fetchData')

// React 组件性能监控
function MyComponent() {
  useRenderMonitor('MyComponent', true)

  return <div>...</div>
}

// 获取性能报告
const report = performanceMonitor.getReport()
console.log(report)
```

### 监控指标

- 渲染时间 (render)
- 长任务 (longtask)
- 内存使用 (memory)
- 资源加载 (resource)
- 导航时机 (navigation)
